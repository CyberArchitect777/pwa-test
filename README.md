# pwa-test

An experimental Progressive Web App (PWA) built as a test of both PWA technology and AI-assisted coding using [Claude Code](https://claude.ai/code).

## Purpose

This project serves two experimental goals:

1. **PWA exploration** — testing how a web app can be installed to a device home screen, work offline, and behave like a native app using a web manifest and service worker.
2. **AI coding demonstration** — the app and its PWA infrastructure (manifest, icons, service worker) were written with Claude Code as a demonstration of AI-assisted development.

## The App

A simple dice roller. Tap the button, get a number between 1 and 6.

## Live App

[https://cyberarchitect777.github.io/pwa-test/](https://cyberarchitect777.github.io/pwa-test/)

Visit the link on a mobile device and use "Add to Home Screen" to install it as a PWA.
